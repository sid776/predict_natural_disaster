Quantum Natural Disaster Prediction System

About the Project

This project uses quantum-inspired algorithms and AI to predict probabilities of various natural disasters (tornados, earthquakes, forest fires, and floods) for locations worldwide.

View Documentation
Try the Application

To run the application locally, follow these steps:

  1. Clone the repository
  2. Install dependencies: pip install -r requirements.txt
  3. Run the application: python app.py
  4. Open your browser to http://localhost:5000
